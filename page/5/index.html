新手学习Angularjs之二：ng-bind、ng-model、大括号的区别

本文假设nodejs及bower环境变量已经设置完成，如果还没有设置，可参见这里：windows下搭建node.js及npm的工作环境（更新版）、Node.js的项目创建及依赖维护工作。

在学习的过程中，遇到了这三个对象，一时把我给搞糊涂了。经过百度，得到了以下答案：

  • ng-model：用于双向绑定，所以一般用于前台可输入的控件，如input。
  • ng-bind：只用于前台展示，所以一般和div结合使用。
  • 双大括号：其基本作用类似ng-bind，但不同的是，在angularjs未加载完成时，由于页面未解释，所以很有可能客户会看到大括号，所以一般<div ng-clock class="ng-clock">｛{uname}｝</div>这样使用，这样可使页面先不加载大括号，待全部加载完成后再解释执行。

svg中节点顺序的替代解决方案

如果你的前端页面图形是使用svg画的，那么有一个问题很有可能遇到的，就是节点间先后顺序的问题。由于svg元素没有z-index属性，画的顺序决定了显示的顺序，所以当你拖拽一个节点时，很容易会发生，该节点与其他节点重合时会跑到其他节点下面的尴尬。

###raphaeljs下解决方案
如果你使用的是raphaeljs，那么恭喜你，这都不是事，raphaeljs中提供了如下方法，直接操作就可以了！

element.toBack()
element.toFront()

当然，如果你没有使用svg，那么悄悄的把raphaeljs用起来也不错哦。

Linux折腾记(2)-CentOS6下匿名共享可读写文件夹

想把windows上的一些资料定时备份到linux，首先想到的就是利用共享文件夹的方式，折腾了一番，最终还是搞成功了。过程如下：

  1. 首先安装samba及samba-client,中间不管提问什么，一路y即可。
    yum install samba samba-client
  2. 在home下创建文件夹test。
    mkdir /home/test

  3. 将/etc/samba/smb.conf文件中的内容替换成。

    [global]
security=share
dos charset=cp936
unix charset=gb2312

[mygitback]
comment = share
path = /home/test
browseable = yes
guest ok = yes
writable = yes
