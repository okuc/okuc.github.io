shell学习(三)

shell中的输入输出重定向

  • 标准输入是键盘，标准输出是终端显示器。
  • 输出重定向
    • command > outputfile //大于号两端要有空格分开
    • command >> outputfile

  • 一个例子

    echo 22222 > ./iotest
echo 111 > ./iotest //会把上一行覆盖掉
echo 111 >> ./iotest //不会覆盖掉原有数据，会新起一行
  • 输入重定向
    • command < outputfile
  • 例子： cat < iotest > iotest2

  • 内联输入重定向

    command << marker
data input
marker

  • 例子：

    result = `bc << EOF
>var1=3
>var2=4
>var3=var1+var2
>print var3
>EOF`

echo $result

shell学习(二)

shell编程中字符串的操作

  • 连接字符串

    str=hello
str1=world
连接起来：str=$str$str1、str = $str' '"aa123"、str = $str:

  • 查看字符串长度：

    • expr length "hello"
    • expr length "$str"
    • charcount= ` expr length “$str”` //反引号，执行里面的代码
    • echo $charcount
  • 查看特定字符：
    • expr index "$str" 'h' #从1开始，不是0，只返回第一个。如果是查找多个字母，只返回这些字母的第一个查找到的位置
  • 字符串截断：
    • expr substr "$str" 6 10 #获取指定位置指定长度的字符串
  • 正则匹配
    • expr $str : REGEXP
    • expr match $str REGEXP
    • expr $str : '.*' #返回长度
    • expr $str : '\(.*[0-9]\{5\}\)'
    • expr $str : '\([a-zA-X: ]*\).*'

centos6下共享文件

1、首先安装smb服务：yum install samba
2、将配置文件/etc/samba/smb.conf文件修改为：

[global]
security=share
dos charset=cp936
unix charset=gb2312

[szmacs]
comment = share
path = /data/macs
browseable = yes
guest ok = yes
writable = yes

3、关闭访火墙iptables -F
4、关闭selinux：setenforce 0
5、重启smb服务：service smb restart
6、设置smb服务为随机启动：chkconfig smb on

JAVA8学习（一）

lambda表达式

  • 第一个lembda表达式：(String first,String second)->Integer.compare(first.length,second.length);
    • 如果负责计算的代码无法用一个表达式表示，则可以用大括号括起来，并写明返回值。如：
      (String first,String second)->{
    if(first.length<second.length) return -1;
    else return 1;
    }
    • 如果没有参数，仍然可以提供一对空的小括号：()->{for(int i=0;i<100;i++) System.out.println(i);}
    • 如果lambda表达式的参数类型是可以被推导的，那么就可以省略它们的类型，如：Comparator<String> com=(first,second)->Integer.compare(first.length(),second.length());,由于它被赋于一个字符串比较器，所以参数必须字符串。

dockerfile指令学习

  • FROM：它必须针对已经存在的镜像，且是基础镜像来操作。而且它必须是第一条非注释指令。
  • 第一个dockfile文件，文件名必须为：Dockerfile,内容如下,然后使用docker build -t okuc/okuc_test .命令执行：
    #First Dockerfile
FROM centos:6.6
MAINTAINER okuc "okuc@live.cn"
RUN yum install -y java
EXPOSE 80
  • MAINTAINER <name>:指定镜像的作者信息，包含镜像的所有者和联系信息。
  • RUN指定当前镜像中运行的命令。
    • RUN (shell模式)
      /bin/sh -c command
RUN echo hello
    • RUN [“executable”,”param1”,”param2”] (exec模式)
      RUN["/bin/bash","-c","echo hello"]
