Linux折腾记(5)-centos7下安装markdown编辑器haroopad-v0.13.1-x64

安装上了windos10和centos7双系统，想找一款比较不错的markdown双系统，找来找去，最终决定安装haroopad，安装过程如下：

1.首先是下载。我安装的目前最新版本haroopad-v0.13.1-x64.tar.gz,下载地址如下：haroopad-v0.13.1-x64.tar.gz。
如果想安装其他版本，可以在此查找下载更多的安装包：https://bitbucket.org/rhiokim/haroopad-download/downloads/。

2.下载完成后，使用如下命令解压tar -zxvf haroopad-v0.13.1-x64.tar.gz。

3.下面就解压出来的data.tar.gz进行再次解压，并释放到合适的地方。注意:下边的几步都需要root权限。

tar -zxvf data.tar.gz
sudo cp －r ./usr /

4.下面就解压出来的control.tar.gz进行解压并运行。

tar zxf control.tar.gz
chmod 755 postinst
sudo ./postinst

5.运行到这一步，就可以在菜单－编程中看到haroopad了。但是为了让它正确的显示图标，我们还要指定正确的图标位置。编辑如下位置:sudo vi /usr/share/applications/Haroopad.desktop,将文件内容中的Icon，换成正确的图标路径，即可。

[Desktop Entry]
Name=haroopad
Version=0.13.1
Exec=haroopad
Comment=The Next Document processor based on Markdown
Icon=/usr/share/icons/hicolor/128x128/apps/haroopad.png
Type=Application
Terminal=false
StartupNotify=true
Encoding=UTF-8
Categories=Development;GTK;GNOME;
