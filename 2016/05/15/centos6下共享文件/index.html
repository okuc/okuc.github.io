centos6下共享文件

1、首先安装smb服务：yum install samba
2、将配置文件/etc/samba/smb.conf文件修改为：

[global]
security=share
dos charset=cp936
unix charset=gb2312

[szmacs]
comment = share
path = /data/macs
browseable = yes
guest ok = yes
writable = yes

3、关闭访火墙iptables -F
4、关闭selinux：setenforce 0
5、重启smb服务：service smb restart
6、设置smb服务为随机启动：chkconfig smb on
