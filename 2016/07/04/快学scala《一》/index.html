快学scala一

1、在Scala中，每个表达式都有返回值和类型val s= if(x>0) 1 else -1

2、所有的类都继随于Any， 空类型为Unit，写作()

3、语句终止不需要有分号，语句较长的话，要确保第一行以一下不能用作语句结尾的符号结尾。如`+。

4、｛｝块包含一系列的表达式。块中最后一个表达式的值就是块的值。{r=rn;n-=1}最后是一个赋值操作，所以这个块的返回值为Unit。java中赋值操作是有返回值的，返回值是这个赋的值

5、print(“Answer:”+42),println(“aaa”);还可以printf(“Hello,%s!You are %d years old.\n”,”Fred”,42)
6、读取控制台输入：

val name = readLine(“Your name:”)
print(“Your age:”)
val age = readInt() //readDouble,readByte,reaShort,readLong,readFloat,readBoolean,readChar
7、循环：

while(n>0)
r=rn
n-=1

for(i<-0 to n) //包含n
r=r*i

for(i<-0 util n) //不包含n
r=r*i

var sum =0
for(ch<-“Hello”) sum+=ch //将字符串的ASCII码相加

val s=”Hello”
var sum=0;
for(i<-0 util s.length)
sum +=s(i) //结果和上个循环相同

8、scala并没有提供break或continue语句来退出循环。需要break时可以有以下几种选项：

  • 使用Boolean型的控制变量
  • 使用嵌套函数—你可以从函数当中return

  • 使用Breaks对象中的break方法：

    import scala.util.control.Breaks._
    breakable{
    for(…){
    if(…)break;//退出breakable块
    …
    }
    }

9、高级循环

for(i<-1 to 3;j<-1 to 3) print((10*i+j)+” “) //将打印11 12 13 21 22 23 31 32 33

for(i<-1 to 3;j<-1 to 3 if i!=j) print((10*i+j)+” “) //将打印12 13 21 23 31 32

for(i<-1 to 3;from =4-i;j<-from to 3) print((10*i+j)+” “) //将打印13 22 23 31 32 33

for(i<-1 to 10) yield i%3 //生成Vector(1,2,0,1,2,0,1,2,0,1)，for推导式，yield用来构造出一个集合，for推导式与它的第一个生成器是类型兼容的。如下：
for(c<-“Hello”;i<-0 to 1)yield (c+i).toChar //生成HIeflmlmop
for(i<-o to 1;c<-“Hello” )yield (c+i).toChar //生成Vector(’H‘,’e‘,’l‘,’l‘,’o‘,’I‘,’f‘,’m‘,’m‘,’p‘)

10、函数：

def abs(x:Double) = if(x>=0) x else -x //必须给出所有参数的类型，只要函数不是递归的，就不需要指定返回类型
def fac(n:Int)={
var r=1
for(i<-1 to n) r=ri
r //最后一个表达式的值就是函数的返回值
}
def fac(n:Int):Int = if(n<=0) 1 else nfac(n-1) //递归函数必须指定返回类型

11、带名参数：即有些参数如果不传入值，则使用默认值：def decorate(str:string,left:String=”[“,right:String=”]”)=left+str+right //后两个参数lfet和right带有默认值“[“、”]“;还可以在传入的时候指定参数名，这个时候参数顺序就无所谓了：decorate(left=”<<<”,str=”hello”,right=”>>>”)。默认参数和带名参数可以混用。

12、变长参数：

def sum(args:Int)={
var result =0;
for(arg<-args)result+=arg
result
}
val s = sum()
val s =sum(1,4,9,16,25)//参入类型为Seq的参数。
val s=sum(1 to 5)//错误，参数是不是区间，不是参数序列
val s=sum(1 to 5:_)//正确，告诉编译器，这个参数当参数序列处理

//一个递归的例子
def recursiveSum(args:Int):Int={
if(args.length==0) 0
else args.head +recursiveSum(args.tail:_)//args.head为第一个参数args.tail为所有参数
}

当调用变长函数且参数类型为Object的Java方法，如PrintStream.printf或MessageFormat.format时，需要手工对基本类型进行转换。如：val str = MessageFormat.format(“The anser to{0} is {1}”,”everything”,42.asInstanceOf(AnyRef));对于任何Object类型的参数都是这样。

14、懒值：当val被声明为lazy时，它的初始化将被推迟，直到我们首次对它取值。

val words =scala.io.Source.formFile(“/usr/share/dict/words”).mkString //在words被定义时即被取值
layz val words =scala.io.Source.formFile(“/usr/share/dict/words”).mkString//在words被首次使用取值
def words =scala.io.Source.formFile(“/usr/share/dict/words”).mkString//在每一次words被使用时取值

15、受检异常在编译期被检查，需要在声明方法时就抛出异常。scala不支持此种方式。scala中表达式有特殊的类型Nothing,这在if/else表达式中很有用。如果一个分支的类型是Nothing，那么if/else表达式的类型我们就称为是另一个分支的类型。如下例子：

if(x>=0) {
sqre(x)
}else throw new IllegalArgumentException(“x should not be negative”)

上述第一个分支是double，第二个分支类型是Nothing，所以if/else表达式的类型是Double
捕获异常的语法通常采用的是模式匹配的语法：

try{
process(new URL(“http://horstmann.com/fred-tiny.gif“))
}catch{

try{

}finally{
