JAVA8学习（一）

lambda表达式

  • 第一个lembda表达式：(String first,String second)->Integer.compare(first.length,second.length);
    • 如果负责计算的代码无法用一个表达式表示，则可以用大括号括起来，并写明返回值。如：
      (String first,String second)->{
    if(first.length<second.length) return -1;
    else return 1;
    }
    • 如果没有参数，仍然可以提供一对空的小括号：()->{for(int i=0;i<100;i++) System.out.println(i);}
    • 如果lambda表达式的参数类型是可以被推导的，那么就可以省略它们的类型，如：Comparator<String> com=(first,second)->Integer.compare(first.length(),second.length());,由于它被赋于一个字符串比较器，所以参数必须字符串。

dockerfile指令学习

  • FROM：它必须针对已经存在的镜像，且是基础镜像来操作。而且它必须是第一条非注释指令。
  • 第一个dockfile文件，文件名必须为：Dockerfile,内容如下,然后使用docker build -t okuc/okuc_test .命令执行：
    #First Dockerfile
FROM centos:6.6
MAINTAINER okuc "okuc@live.cn"
RUN yum install -y java
EXPOSE 80
  • MAINTAINER <name>:指定镜像的作者信息，包含镜像的所有者和联系信息。
  • RUN指定当前镜像中运行的命令。
    • RUN (shell模式)
      /bin/sh -c command
RUN echo hello
    • RUN [“executable”,”param1”,”param2”] (exec模式)
      RUN["/bin/bash","-c","echo hello"]

TinkerPop3 API 简明笔记（一）

  1. Lambda:尽量避免使用，防止死循环。

    • map：g.V(1).out().map {it.get().value('name')} 等同于 g.V(1).out().values('name')，查询标识为1的节点所指向的节点的名字。
    • filter:过滤。g.V().filter {it.get().label() == 'person'} 等同于 g.V().hasLabel('person'),查找类型（标签）为person的节点。
    • sideEffect:执行某些操作，并把源数据传递给下一步。g.V().hasLabel('person').sideEffect(System.out.&println) 。
    • branch相当于if操作，g.V().branch(values('name')).option('marko', values('age')). option(none, values('name'))等同于g.V().choose(has('name','marko'),values('age'),values('name'))，如果名称为marko，则获取出年龄，否则执行第二步，获取名字。

第 4 章Bootstrap支持的JavaScript插件(2)

  1. 提示条可以用来给出链接说明或（与标签连用）给出缩写词的的全称。这个插件最初的原型是Jason Frame的jQuery.tipsy插件。但
    目前这个提示条利用Bootstrap JavaScript API可以不使用图片，而且有CSS动画。

    用法给<a>元素添加rel="tooltip"就可以添加提示条，<a>元素的title属性值就是提示条的内容。下面两个例子分别使用Bootstrap Data API和JavaScript来创建提示条：

    <a href="#" rel="tooltip" title="This is the tooltip">Tooltip Example</a>
$('#example').tooltip(options)

    选项与其他插件一样，提示条也可以通过Data API或 JavaScript来设置。只不过在使用数据属性时，选项名称前面要加上data-，比如title就要变成data-title。
