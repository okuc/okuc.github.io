shell学习(四)

命令行参数

  • 参数之前用空格分隔
  • 位置参数：$+position:$0,$1,$2....$9
    • $0 :程序名
    • $1,$2....$9
    • ${10} //大于10的需要如此处理

  • basename:获取脚本的名字

    #! /bin/bash
echo `basename $0` //输出程序的直实名字，且不包含路径,即取最后一层的名字

  • 通过联接，把一个脚本建两个连接，从而实现不同的逻辑。下面就是一个例子：
    ln -s calc add、ln -s calc minus

    #! /bin/bash

if [ $# -lt 2 ]  //验证参数
then
    echo 'Please input at least 2 param'
    exit
fi

name=`basename $0`

if [ $name = "add" ]
then
    result=$[ $1 + $2 ]
elif [ $name = "minus" ]
then
    result=$[ $1 - $2 ]
fi
echo "The $name result is $result "

  • 参数数量：$#

  • 所有参数：$* //把所有参数作为一个数来处理

  • 全部参数：$@//把所有参数作为例表处理

    #! /bin/bash

echo $#
echo $*
echo $@

for var in "$*"  //只有一个参数，会报错
do
    echo "\$* Param = $var"
done

for var in "$@"
do
    echo "\$@ Param = $var" //会依次输出$@ Param = XX
done

  • shift 将第一个参数去掉，后面的参数依次向前走一步。

    #! /bin/bash
result=0
while [ -n "$1" ]
do
    result=$[ $result +$1 ]
    shift
done
echo "sum of numbers is: $result"

  • shell命令中选项的使用：

    • 简单实现

      #! /bin/bash

while [ -n "$1" ]
do
    case "$1" in
    -a) echo 'is -a';;
    -b) echo 'is -b';;
    -c) echo 'is -c';;
    esac
    shift
done

    • 复杂实现，分离选项和参数

      #! /bin/bash

while [ -n "$1" ]
do
    case "$1" in
    -a) echo 'is -a';;
    -b) echo 'is -b';;
    -c) echo 'is -c';;
    --) shift
        break; //结束判断
    esac
    shift
done

echo "Params are : $*" //打印出剩下的参数

    • 处理带值的选项

      #! /bin/bash
set -- `getopt -q ab:c "$@"`  //set --意思是将结果赋值给命令行，-q是静默执行的意思，不输出错误
while [ -n "$1" ]
do
    case "$1" in
    -a) echo 'Option -a' ;;
    -b) value="$2"
        echo "Option -b ,Value is : $value"
            shift;;  //处理完后，将值也去掉
    -c) echo 'Option -c' ;;
  --) shift
        break;;
    esac
    shift
done

echo "Params are : $*"

  • getopt 对输入的参数进行格式化，如合并了两个选项，则可以通过它进行格式化，它不能解析包含空隔的选项值

    • getopt abcd -b -acd会格式化为-b -a -c -d --
    • getopt abcd -b -acd param1 param2会格式化为-b -a -c -d -- param1 param2
    • getopt ab:cd -b value -acd param1 param2会格式化为-b value -a -c -d -- param1 param2，:意思是前边的选项是有值的

  • getopts，更高级的参数解析

     #! /bin/bash

 while getopts ab:c opt
 do
   case "$opt" in
   a) echo 'Option -a' ;;
   b) echo "Option -b ,Value is : $OPTARG" //optarg对应选项参数值
             shift;;  //处理完后，将值也去掉
   c) echo 'Option -c' ;;
   *) echo "其它选项： $OPT"
   esac
 done
shift $[$OPTIND -1] //getopts的计数器

count=1
for param in "$@"
do
    echo "param $count is:$param"
    count=$[$count=1]
done
  • shell中一些通用的选项，我们通常用他们来表示相似的函义
    • -a 显示所有对象
    • -c 生成一个计数
    • -d 指定一个目录
    • -e 扩展一个对象
    • -f 指定读入数据的文件
    • -h 显示命令的帮助信息
    • -s 以安静模式运行
    • -v 生成详细输出
    • -x 排除某个对象
    • -y 对所有的问题回答yes

-使用read命令获取用户的输入

  • 基本读取,将输入的数字放到param中： read param

  • 设置5S超时： if read -t 5 -p “这是提示” input 

    #! /bin/bash

      if read -t 5 -p "Please type your input:" input
      then
          echo "You typed: $input"
      else
          echo "Timeout"
      fi

- 隐藏输入的文字 ,如密码，`-s`选项可以做到。

- 从文件中读取,通过输入重定向和管道两种方式


        #! /bin/bash

        exec 0< read_test1
        count=1

        while read line
        do
            echo "#$count: $line"
            count=$[ $count + 1 ]
        done


        #! /bin/bash

        count=1
        cat read_test1 | while read line
        do
            echo "#$count: $line"
            count=$[ $count + 1 ]
        done
