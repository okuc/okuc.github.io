centos7常用命令

记录centos7下的一些常用命令!

###网络相关

  • 关闭防火墙：
    systemctl stop firewalld.service#停止firewall
systemctl disable firewalld.service#禁止firewall开机启动

  • 配置DNS服务

    • 命令行方式

      1. nmcli connection show #显示当前网络连接
NAME UUID                                 TYPE           DEVICE
eno1 5fb06bd0-0bb0-7ffb-45f1-d6edd65f3e03 802-3-ethernet eno1

2. nmcli con mod eno1 ipv4.dns "114.114.114.114 8.8.8.8" #修改当前网络连接对应的DNS服务器，这里的网络连接可以用名称或者UUID来标识
3. nmcli con up eno1#将dns配置生效
    • 直接修改文件方式 
      1. 修改 /etc/NetworkManager/NetworkManager.conf 文件，在main部分添加 “dns=none” 选项：
    [main]
    plugins=ifcfg-rh
    dns=none
2. NetworkManager重新装载上面修改的配置
    systemctl restart NetworkManager.service
3. 手工修改 /etc/resolv.conf
    nameserver 114.114.114.114
    nameserver 8.8.8.8

  • 配置网关、IP等

    vim /etc/sysconfig/network-scripts/ifcfg-enp0s3

BOOTPROTO="static" #dhcp改为static
ONBOOT="yes" #开机启用本配置
IPADDR=192.168.7.106 #静态IP
GATEWAY=192.168.7.1 #默认网关
NETMASK=255.255.255.0 #子网掩码
DNS1=192.168.7.1 #DNS 配置

service network restart #重启网络服务

ip addr #查看网络地址信息

    注意：文件名称不一定是ifcfg-enp0s3,需要自行去文件目录下查看

  • 修改本地host文件
    vi /etc/hosts
192.168.90.135  www.okuc.xyz

###网络相关

  • 修改机器名：
    • 命令行方式
      hostname 主机名 #临时生效，重启后恢复原来机器名
      hostnamectl set-hostname #永久生效
    • 文件方式
      vi /etc/sysconfig/network
      NETWORKING=yes
      HOSTNAME=www.okuc.xyz
