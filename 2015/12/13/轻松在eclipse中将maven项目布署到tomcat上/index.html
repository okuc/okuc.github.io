轻松在eclipse中将maven项目布署到tomcat上

在eclipse中导入已经建好的maven项目后，往往会出现无法布署到tomcat上的情况，此时，该怎么办呢？

  1. 右键项目名，选择Properties。在弹出的页面中，左侧选择Project Facets,在右侧的列表中，选中Dynamic Web Module、Java、JavaScript三项，然后点击右下角的Apply按扭。

  2. 左侧选择Deployment Assembly,在左则，配置好依赖：

    • /src/main/java–>WEB-INF/classes
    • /src/main/resources–>WEB-INF/classes
    • /src/main/webapp–>/
    • /src/test/java–>WEB-INF/classes

    • Maven Dependencies–>WEB-INF/lib

      然后点击右下角的Apply按扭。

  3. 点击OK按扭返回，这回，可以直接在项目名上点击右键，选择Run AS -> Run on Server了。
