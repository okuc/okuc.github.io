Emmet常用缩写简要笔记

emmet可谓前端开发神器，特别是在前端页台和后端服务分离化，restful大肆流行的今天，掌握点emmet还是有必要的。现将一些常用知识点简单记录如下。

  1. 初始化文档。

    • html:5 或!：用于HTML5文档类型。生成的代码如下，注意,生成后需要把lang="en"改为lang="zh-cn"，再把’title’改成相对应的文档标题就可以了。

      <!DOCTYPE html>
<html lang="en">
<head>
    <meta charset="UTF-8">
    <title>Document</title>
</head>
<body>

</body>
</html>
    • html:xt：用于XHTML过渡文档类型。
    • html:4s：用于HTML4严格文档类型。
  2. 子代：>用来表示子代。代码div>ul>li可以生成的代码：
    <div>
    <ul>
        <li></li>
    </ul>
</div>
  3. 兄弟：+用来表示兄弟。代码div+p+bq可以生成的代码：
    <div></div>
<p></p>
<blockquote></blockquote>

  4. 父代：^用来表示父代。代码div+div>p>span+em^bq可以生成的代码：

    <div></div>
<div>
    <p><span></span><em></em></p>
    <blockquote></blockquote>
</div>

  5. 重复：*用来表示重复。代码ul>li*5可以生成的代码：

    <ul>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
    <li></li>
</ul>
  6. 成组：()用来表示成组。代码(div>dl>(dt+dd)*3)+footer>p可以生成的代码：
    <div>
    <dl>
        <dt></dt>
        <dd></dd>
        <dt></dt>
        <dd></dd>
        <dt></dt>
        <dd></dd>
    </dl>
</div>
<footer>
    <p></p>
</footer>
  7. ID:# #用来表示ID。代码div#header+div.page+div#footer.class1.class2.class3可以生成的代码：
    <div id="header"></div>
<div class="page"></div>
<div id="footer" class="class1 class2 class3"></div>

  9. 属性：[]用来表示ID。代码td[title="hello world!" colspan=3]可以生成的代码：
    <td title="hello world!" colspan="3"></td>
